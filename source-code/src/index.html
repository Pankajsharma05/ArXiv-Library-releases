Welcome to ArXivLibrary ✦

A quick tour of what you can do:

⌕ Search & save — search arXiv, then save papers to your library. Drag results onto a collection or tag to file them instantly.
✦ Daily Feed — follow categories like cond-mat.supr-con to see the newest papers each day.
⊛ Graph — connect related papers with labelled arrows and explore your library visually.
☷ Organize — collections, colored tags, reading status, and Markdown notes. Right-click any paper for actions, or multi-select with checkboxes.
⌨ Tips — ⌘F to filter, ↑/↓ to navigate, collapse panels from the header/corner, and find time stats + feedback in Settings.

About
ArXivLibrary
Version 1.0

An arXiv-native research library — search, organize, annotate, and connect papers.

Developed by Pankaj Sharma.
Built with Tauri, Rust, and Claude (Anthropic).

This project uses the arXiv API. Thank you to arXiv for use of its open access interoperability. ArXivLibrary is an independent project and is not affiliated with or endorsed by arXiv.

With thanks to the open scholarly infrastructure that makes this possible: Semantic Scholar (citation data and DOI resolution), Crossref (DOI metadata lookup), and the DOI system. ArXivLibrary is independent and not affiliated with or endorsed by any of these services.
